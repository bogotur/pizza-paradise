# 🍕 Pizza Paradise

Fullstack web application for pizza delivery with a modern frontend and scalable backend architecture.

This project was built as a **portfolio project** to demonstrate fullstack development skills using **TypeScript** on both client and server sides.

---

## 🚀 Tech Stack

### Frontend
- Next.js
- React
- TypeScript
- CSS Modules

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- MySQL
- phpMyAdmin

---

## ✨ Features

- Pizza catalog with categories
- Client–server architecture
- REST API
- TypeScript on frontend and backend
- Clean and scalable project structure
- Ready for further extension (authentication, orders, payments)

## ⚙️ Installation & Run

### 1️⃣ Clone repository

```bash
git clone https://github.com/bogotur/pizza-paradise.git
cd pizza-paradise
2️⃣ Frontend setup
cd client
npm install
npm run dev
Frontend will be available at:

http://localhost:3000
3️⃣ Backend setup
cd server
npm install
npm run dev
Backend API will run on:

http://localhost:5000

🗄️ Database

MySQL is used as the main database

phpMyAdmin is used for database management

Database structure can be stored in database/schema.sql
